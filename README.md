## Notex Application Backend 

-This is the NoteX web application backend code that develop using node.js, express.js and mongodb <br>
-All CRUD operation are performe here <br>
-Authetication system are build using JWT token, bytcrypt methods 

## Setup 

```npm
git clone "https://github.com/TejasGtech/Notes-Backend-Code.git"
cd backend
```

## install

```bash
npm install
npm start
```

Then the bacekend are start succefully!
